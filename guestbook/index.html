A PUBLIC EXPERIMENT

One deck. Everyone.

The Guestbook is a single presentation file that everyone edits at once, live. There is no login and no server that can read it — the encryption keys travel inside the file, so opening it is joining it. Leave a note on a wall, watch strangers' cursors wander, take a copy home.

  • Add, don't wipe. Be kind. Anything goes except cruelty.
  • It resets on a schedule — fresh keys, blank walls. Beautiful epochs get archived; everything else evaporates.
  • Your saved copy keeps working forever, offline. That's the product.

Under the hood this is stock Bento collaboration — E2EE (AES-GCM), a blind relay that stores ciphertext, and a CRDT that merges everyone's edits. Nothing special was built for this page. Set your name via the Live button so your cursor has one.
