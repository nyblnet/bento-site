01
Everything ships inside

Software with nothing to install, break, or expire.

No account. No cloud. No “please update to open this”. A Bento deck is an ordinary HTML file that carries its own viewer, presenter and editor — and your document, as readable JSON you can open in any text editor.

View-source honest

Your data sits in a plain, legible block at the top of the file. No binary formats, no lock-in — ever.

Fonts & assets travel

Typefaces, images and charts are embedded. A deck looks the same on any machine, forever.

Any copy opens alone

Email it, AirDrop it, archive it for a decade. Every copy is the complete product.

Updates you apply

Cryptographically signed releases, offered in-app. You choose when — and the old file stays as rollback.

03
Local-first · AI-native

Designed for AI.
Entirely optional.

Bento is local-first and built for the AI era. The document is plain JSON sitting openly in the file — so a chatbot round-trips it and an agent harness edits the file directly. That's how the gallery above was made. No converters, no uploads, no .pptx archaeology: orders of magnitude leaner than driving PowerPoint with a bot. Bringing an old deck across? Hand the .pptx to any AI alongside bento.page/agents.md and ask it to rebuild — you get real Bento slides (morph, live charts), re-authored, not lossily imported. And if you never touch AI, nothing here ever will either.

The format is the API No binary .pptx to decode Agents edit the file in place bento.page/agents.md — the recipe for your agent Nothing phones home You choose the AI to use

Morph.

Elements that share an id animate between slides — position, size, color, even gradients. Duplicate a slide, move things around, and the motion designs itself.

07
Local-first, provably

No cloud required.
Not even ours.

Flip on Offline mode and nothing ever leaves your computer — updates and online collaboration are hard-blocked, and the app says so in plain words. The file is the durable artifact; everything else is optional convenience.

Yours to keep

Decks survive us. If bento.page vanished tomorrow, every file keeps opening, editing and presenting.

Open source runtime

MIT-licensed components throughout — reveal.js, Moveable, our own chart and animation engines.

No telemetry

Update checks send nothing about you or your document. No ids, no analytics, no phoning home.

One file.
Yours. Forever.
